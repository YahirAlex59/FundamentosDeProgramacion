# FundamentosDeProgramacion
Repositorio de actividades de programacion de la materia *Fundamentos de Programación*.  
Cada archivo contiene la solución a un problema planteado en clase.  

### *Actividad 1*
📄 Archivo: EstructurasSelectivasAct1.java
📝 Programa que dice el numero de dias de cada mes

### *Actividad 2*
📄 Archivo: ACTIVIDAD2SwitchCase.java  
📝 Programa que lee Realiza *suma, resta, multiplicación o división*.   

### *Actividad 3*
📄 Archivo: ACTIVIDAD3SwitchCase.java  
📝 Programa con un *menú de cálculo de áreas*.  
Opciones:  
- Cuadrado  
- Rectángulo  
- Triángulo  
- Círculo  

### *Actividad 4*
📄 Archivo: ACTIVIDAD4SwiftCase.java  
📝 Programa Buenos dias,tardes...

### *Actividad 5*
📄 Archivo: ACTIVIDAD5SwitchCase.java  
📝 Programa que muestra el *horóscopo* correspondiente al dia y mes


### *Actividad 6*
📄 Archivo: ACTIVIDAD6SwitchCase.java  
📝 Programa que calcula el *precio de entradas de cine*.   


### *Actividad 7*
📄 Archivo: ACTIVIDAD7SwitchCase.java  
📝 Programa *pastelería*
